# SATCCheckLists - Система электронных чек-листов

Это приложение представляет собой систему электронных чек-листов с возможностью голосовой авторизации.

## Основные функции

*   **Авторизация:** Вход в систему по логину и паролю.
*   **Голосовая авторизация:** Возможность входа с помощью голосовой команды, произнеся 4-значный пароль.
*   **Фильтрация пользователей:** Быстрый поиск нужного пользователя по категориям "Вышка" и "Зал".
*   **Электронные чек-листы:** (Основной функционал приложения, который будет доступен после авторизации).

## Инструкция по использованию

### Обычная авторизация

1.  **Выберите пользователя:** В выпадающем списке найдите и выберите свое имя.
2.  **Введите пароль:** В поле для пароля введите ваш 4-значный цифровой пароль.
3.  **Нажмите "Войти":** Если данные верны, вы войдете в систему.

### Голосовая авторизация

1.  **Нажмите на иконку микрофона:** Кнопка находится справа от поля ввода пароля.
2.  **Дайте разрешение на использование микрофона:** При первом использовании браузер запросит доступ к вашему микрофону. Нажмите "Разрешить".
3.  **Произнесите пароль:** Четко произнесите ваш 4-значный пароль в течение 3 секунд.
4.  **Автоматический вход:** Если пароль распознан и найден в системе, приложение автоматически подставит ваши данные и выполнит вход.

### Возможные ошибки

*   **"Пароль не найден"**: Убедитесь, что вы четко произнесли пароль. Попробуйте еще раз.
*   **"Доступ к микрофону запрещен"**: Проверьте настройки вашего браузера и разрешите доступ к микрофону для этого сайта.
*   **"Ошибка сети"**: Проверьте ваше интернет-соединение. Для распознавания речи требуется стабильный доступ к сети.
